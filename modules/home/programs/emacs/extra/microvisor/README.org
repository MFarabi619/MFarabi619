* Description :unfold:
This module adds support for embedded systems development with:
This module enhances the Emacs experience for Embedded Systems Development,
and also provides a united interface for overall project management, tasks
such as testing, continous integration, continous deployment, etc.

It does this with several modular packages focused on enhancing the built-in
~compile~ interface. The primary packages are:

- Fancy Compilation, which provides beautiful colored outputs
- Compile Multi, which provides a supercharged compilation experience with live narrowing and fuzzy finding
- Prodigy, which provides service management primitives with processes and ports
- Projectile, which provides workspace discoverty and deep integration with existing build systems
