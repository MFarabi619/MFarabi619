**** Other
ESP32-S3 flash is 8MB, divided into a dual-OTA layout for safe over-the-air updates.
The bootloader checks =otadata= to decide whether to boot from =ota_0= or =ota_1=.
A new firmware image is written to the *inactive* slot, then =otadata= is flipped.
If the new image fails to boot, the bootloader rolls back automatically.

| Partition | Offset   | Size  | Purpose                                           |
|-----------+----------+-------+---------------------------------------------------|
| =nvs=       | =0x9000=   | 16 KB | Non-volatile storage (WiFi credentials, settings) |
| =otadata=   | =0xD000=   | 8 KB  | Tracks which OTA slot is active                   |
| =phy_init=  | =0xF000=   | 4 KB  | RF calibration data for WiFi/BLE                  |
| =ota_0=     | =0x10000=  | ~4 MB | Primary firmware slot                             |
| =ota_1=     | =0x400000= | 4 MB  | Secondary firmware slot (OTA target)              |
