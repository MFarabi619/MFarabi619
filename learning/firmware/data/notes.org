- =IOC= - Interrupt on Change
  - Feature to detect when a GPIO pin changes state (low‚Üíhigh or high‚Üílow) and trigger an interrupt so the CPU can respond immediately instead of constantly polling the pin.
  - Buttons, switches, or any digital input where you want to react instantly to a change.
  - You configure a pin to generate an interrupt on a rising edge, falling edge, or both edges. When the event occurs, the NVIC (Nested Vectored Interrupt Controller) triggers an interrupt service routine (ISR).
  - Saves CPU cycles because you don‚Äôt need to constantly check pin status. The MCU just ‚Äúwakes up‚Äù when something changes.
  - STM32 HAL, this is typically set up via functions like HAL_GPIO_EXTI_Callback() after enabling the pin‚Äôs EXTI (External Interrupt) line.
